1. **环境搭建**: Rust 1.91+ nightly
2. **代码编写**: 遵循 CLAUDE.md 规范
3. **测试验证**: 单元测试 + 集成测试

5. **部署发布**: 生产环境部署

- **编译器**: rustc 1.91+ nightly
- **构建工具**: cargo
- **测试框架**: cargo test
- **Benchmark**: criterion
- **代码检查**: clippy
- **格式化**: rustfmt
