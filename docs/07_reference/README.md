## 📊 快速参考

### 性能指标
- 订单吞吐: > 100K/s
- 撮合延迟: P99 < 100μs
- WAL 写入: P99 < 50ms
- MemTable 写入: P99 < 10μs

### 容量限制
- 并发账户: > 10,000
- WebSocket 连接: > 10,000
- 日志保留: 30 天
- SSTable 大小: 128 MB/file
